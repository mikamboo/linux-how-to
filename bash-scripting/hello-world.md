# Shebang

> Le shebang, représenté par **\#!**, est un en-tête d'un fichier texte qui indique au système d'exploitation que ce fichier n'est pas un fichier binaire mais un script

* Bash

```bash
#!/bin/bash

# This i a bash script
echo Hello
```

* Shell

```bash
#!/bin/sh

# This i a bash script
echo Hello
```

* Python

```python
#!/usr/bin/python

# This i a bash script
print("Hello")
```
